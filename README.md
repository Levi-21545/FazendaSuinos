# Fazenda de Suínos - PI3

Estrutura basica baseada em SeedProject.NET

### Propósitos das Pastas

- doc
  - Qualquer documentação relevante para o conteúdo do repositório
- lib
  - Dependências de compilação
  - Estas não devem ser do NuGet
- packages
  - Pacotes NuGet
    - Eu sei, eu sei, não deveriam ser incluídos, às vezes um projeto está bagunçado o suficiente para que isso seja necessário durante a limpeza...
- src
  - O código-fonte deste repositório
  - A raiz desta pasta deve consistir apenas de pastas onde quaisquer arquivos sln's ou .\*proj devem estar dentro de pastas de solução ou projeto apropriadas.
- data
  - Os projetos e arquivos de dados para este repositório
  - Projetos de dados (por exemplo, _.sqlproj) devem ser armazenados em diretórios, arquivos de dados (por exemplo, _.json, \*.xml) podem ser armazenados na raiz desta pasta.
